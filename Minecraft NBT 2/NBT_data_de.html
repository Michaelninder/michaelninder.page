Beispiele für NBT-Data in Minecraft

_________________________________.___._.___._________________________________
CommandFunktionNBT
/give @a Stick{Enchantments:[{id:"minecraft:knockback", lvl:255s}]} Gibt allen Spielern einen
Stick, der die Verzauberung
"Knockback1000" hat		 Das NBT-Tag
ist hier die
Verzauberung
/give @a Stick[Tag=Test] Gibt allen soieler einen Stick
der ein tag mit dem Namen
"Test" hat. 		Das NBT-Tag
ist hier das
Tag namens Test
/give @a Sculk_catalyst[Tag=Tp] Gibt allen Spielern einen
Tag Sculk Catalyst
mit dem Tag "Tp"		 Das NBT-Tag
ist hier das
Tag namens Tp
/give @a "blockname"[tag="dein Tag"] Gibt allen Spielern einen
bestimmten Block mit
einem bestimmten Tag		 Das NBT-Tag
ist hier das
bestimmte Tag
/enchant @a minecraft:unbreaking 3 Verzaubert bei allen
spielern den gegenstand,
den sie in der Hand
haben mit haltbarkeit 3		 das NBT-Tag
Verzauberung:
Haltbarkeit 3
wird hinzugefügt
/enchant >bei wem< >Verzauberung< >level<
